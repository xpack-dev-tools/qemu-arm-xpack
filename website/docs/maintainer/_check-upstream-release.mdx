### Check the latest upstream release

The current stable release is expected
to be announced in the [QEMU](https://qemu.org)
web, and reflected in the Git
[tags](https://gitlab.com/qemu-project/qemu/-/tags).

Compare the latest upstream release with the current xPack
[release](https://github.com/xpack-dev-tools/qemu-riscv-xpack/releases).
If necessary, update the triplet, otherwise increase the fourth number,
as below.
