<HistoryGnuArmEclipse date="August 2015"/>

The initial name of the package was
[GNU MCU Eclipse QEMU GCC](https://github.com/ilg-archived/qemu)
and was first published on `npmjs.com` as
[`@gnu-mcu-eclipse/qemu`](https://www.npmjs.com/package/@gnu-mcu-eclipse/qemu).

### April 2015

The first **xPack QEMU Arm** release was
[v2.2.92-20150404](/blog/2015/04/04/qemu-v2-2-92-20150404-released/).

### July 2019

<HistoryRenameXpack/> and the npm scope was changed from `@gnu-mcu-eclipse` to
[`@xpack-dev-tools`](https://www.npmjs.com/package/@xpack-dev-tools/qemu-arm)
(in [v2.8.0-7](/blog/2019/07/22/qemu-arm-v2-8-0-7-released/)).
